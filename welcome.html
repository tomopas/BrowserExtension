Welcome!

Thank you for installing the Random secret based Hash Password Generators (RSHPG). This extension helps you generate and autofill secure passwords.

More Information on the technical implementation can be found on GitHub: https://github.com/tomopas

First you need to generate a Random Value or input one. This Random Value will be stored in your browser and used to generate the password. You can also transfer it to another device and use it to generate the password there.

Click the extension icon to start using it.
